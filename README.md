학업의 순서로는
1. Java언어에 대해서 학습
2. Android_Studio의 환경에 대해서 학습
3. Android_Studio내에서 Java언어를 사용한 여러가지 프로그램을 코딩하고 실행해본다.
4. 최종 프로그램으로는 농장 키우기 게임이나 RPG형태의 게임을 목표로 하고있다.
